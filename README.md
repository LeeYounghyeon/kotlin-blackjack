# kotlin-blackjack

## 기능 목록
- 게임에 참여할 사람의 이름을 입력한다.
    - 쉼표 기준으로 분리한다.
- 게임을 시작하면 플레이어에게 두 장의 카드를 지급한다.
    - 두장의 합이 21을 초과하지 않으면 카드를 뽑을 수 있다.
- 각 사용자가 받은 카드를 보여준다.
    - 숫자와 모양
- 사용자들이 카드를 더 받을지 안받을지 선택한다.
    - y 와 n으로 선택한다.
- 한장의 카드를 더 받으면 사용자가 가지고 있는 카드를 보여준다.
- 두장의 카드를 합쳐 21을 초과하지 않으면서 21에 가까운 사람이 이긴다.
- 우승자가 있으면 결과를 보여준다.
